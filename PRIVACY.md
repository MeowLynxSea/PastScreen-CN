# 隐私政策

PastScreen-CN 是一款本地运行的截图工具。本隐私政策说明我们如何处理你的数据。

## 收集的信息

- 本应用不收集个人身份信息。
- 不包含分析、广告或追踪 SDK。
- 不会创建或上传用户画像。

## 本地数据

- 截图文件与应用设置仅保存在你的设备上。
- 如果你启用了保存到文件，图片会保存在你选择的目录。

## 权限说明

- 屏幕录制：仅用于截图。
- 辅助功能：仅用于全局快捷键。
- 通知：用于截图完成提示。

## 网络与第三方

- 应用核心功能不依赖网络，不会向服务器上传内容。
- 不与第三方共享数据。

## 变更

如有变更，将在本文件中更新。

## 联系方式

如有疑问，请在 GitHub Issues 中联系。
